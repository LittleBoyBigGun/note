# 总览

**setText**(`text`, `index?`): `Promise`<`boolean`>

设置文本

# 参数

| 名称  | 类型   | 默认值    | 描述                              |
| ----- | ------ | --------- | --------------------------------- |
| text  | string | undefined | 设置的文本内容                    |
| index | number | 0         | 输入至屏幕中第几个输入框，从0开始 | 

# 返回值

`Promise`<`boolean`>

是否成功的Promise
