# 总览

**swipe**(`x1`, `y1`, `x2`, `y2`, `duration`): `Promise`<`boolean`>

直线滑动

# 参数

| 名称     | 类型   | 默认值 | 描述               |
| -------- | ------ | ------ | ------------------ |
| x1       | number | tbd    | 起点X坐标          |
| y1       | number | tdb    | 起点Y坐标          |
| x2       | number | tdb    | 终点X坐标          |
| y2       | number | tdb    | 终点Y坐标          |
| duration | number | tdb    | 滑动时长，单位毫秒 | 

# 返回值

`Promise`<`boolean`>

返回是否运行成功的Promise。运行完成时触发。
