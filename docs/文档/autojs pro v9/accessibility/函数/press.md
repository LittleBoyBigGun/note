# 总览

**press**(`x`, `y`, `duration`): `Promise`<`boolean`>

按住屏幕指定位置一段时间。

# 参数

| 名称     | 类型   | 默认值 | 描述               |
| -------- | ------ | ------ | ------------------ |
| x        | number | tbd    | X坐标              |
| y        | number | tbd    | Y坐标              |
| duration | number | tbd    | 按住时长，单位毫秒 | 

# 返回值

`Promise`<`boolean`>

返回是否运行成功的Promise。运行完成时触发
