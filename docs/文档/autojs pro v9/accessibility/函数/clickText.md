# 总览

**clickText**(`text`, `index?`): `Promise`<`boolean`>
点击屏幕中的文本。

# 参数
| 名称  | 类型   | 默认值    | 描述                            |
| ----- | ------ | --------- | ------------------------------- |
| text  | string | undefined | 需要点击的文本内容              |
| index | number | 0         | 点击屏幕中的第几个文本，从0开始 | 

# 返回值

`string` | `null`

Activity名称。无权限则返回null。
