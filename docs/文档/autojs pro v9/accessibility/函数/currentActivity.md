# 总览

**currentActivity**(): `string` | `null`

返回最近一次监测到的正在运行的应用的Activity名称，一般可以认为就是当前正在运行的应用的Activity名称。

# 参数
| 名称 | 类型 | 默认值 | 描述                            |
| ---- | ---- | ------ | ------------------------------- |
|      |      |        |                                 |
|      |      |        |  |

# 返回值

`string` | `null`

Activity名称。无权限则返回null。
